About this Project

This project is a collaboration between several computer science students at Pasadena City College, with varying degrees of experience. This project is primarily to give students a chance to collaborate in a larger group than is typically encountered during classes.

Joining the Project

If you are interested in contributing, please contact the project owner at the email address listed in the GitHub profile at the bottom of this page. In your email, please include the following information (Optional fields are in italics):

Name
Primary Contact Info E-mail address or phone number to text regarding project.
Secondary Contact Info
Experience Experience is not necessary; this is used to figure out where you can best contribute. Please list any CS courses you have completed, are currently enrolled in, or equivalent experience outside coursework.
Availability If you can only meet in person at certain times/on certain days, please let me know. If a meeting is being held, you will be notified by your preferred method.
Areas of Interest/Expertise Any ares you are interested in working on (such as QtGUI, openGL, object design, formula derivation, writing only basic functions, etc.), or where you have relevant experience to offer.

Contributing to the Project
